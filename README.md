# Nivra.DotNetAnalyzers

This repository contains a collection of custom Roslyn analyzers and code fixes for .NET projects. These analyzers help enforce coding standards, improve code quality, and provide automated refactoring suggestions.

Refactor the method definition and invocation to have each parameter and argument on its own line for better readability.
